Esta vista fue retornada usando la clase ModelAndView.

¿Qué es ModelAndView?
Es una clase de Spring MVC que permite retornar en una sola estructura:
  • El nombre de la vista que se debe renderizar
  • El modelo de datos que será enviado a la vista

En el controlador, se crea un objeto ModelAndView especificando el nombre de esta plantilla, y Spring se encarga de resolver automáticamente el archivo HTML correspondiente en la carpeta templates/.
