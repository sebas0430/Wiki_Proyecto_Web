Proyecto / Visión General

Editor de Procesos Empresariales

Semestre Actual • Desarrollo Web

El proyecto consiste en el desarrollo de un visor y editor de procesos empresariales. Su objetivo es permitir la visualización y edición de procesos asociados a una empresa, garantizando la separación de información entre distintas organizaciones (multi-tenancy).

Alcance del Sistema

El sistema estará orientado únicamente a la visualización y edición, no a la ejecución de los procesos.

Enfoque Académico: Los usuarios podrán consultar, crear, modificar y organizar procesos, pero no se dispararán flujos automáticos ni tareas de procesamiento real. El objetivo es evaluar arquitectura, seguridad y buenas prácticas.

Plan de Entregas

El desarrollo se realizará de forma incremental en tres etapas:

1. Backend (Lógica de Negocio)

Implementación de servicios REST para gestionar empresas, usuarios y procesos. Diseño del modelo de datos y lógica fundamental.

2. Frontend (Interfaz Web)

Construcción de la interfaz gráfica que consuma los servicios del backend, permitiendo la interacción visual con los procesos.

3. Integración Final

Despliegue completo del sistema (Frontend + Backend), asegurando seguridad, calidad de código y funcionamiento de extremo a extremo.
